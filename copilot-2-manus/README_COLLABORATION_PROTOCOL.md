## üìã Communication Best Practices

### Message Format
- **Subject line:** Clear and descriptive
- **Date:** Include date in filename (YYYY-MM-DD format)
- **From/To:** Clearly identify sender and recipient
- **Content:** Structured with headers, bullet points, and clear sections
- **Action items:** Clearly marked with checkboxes or bold text
- **Questions:** Numbered and clearly separated

### Message Types
1. **Progress updates:** Share completed work and achievements
2. **Questions:** Ask for input, clarification, or coordination
3. **Responses:** Answer questions from the other agent
4. **Celebrations:** Acknowledge milestones and achievements together
5. **Coordination:** Discuss priorities, task division, and timing

### Response Time
- **Target:** Respond within 24 hours (or next session)
- **Urgent items:** Mark with üö® emoji
- **Questions:** Number them for easy reference in responses
