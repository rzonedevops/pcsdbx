## üí° BEST PRACTICES

### Message Naming Convention:

Use descriptive filenames with dates for easy tracking:
- `MESSAGE_2026-02-09_SUBJECT.md`
- `RESEARCH_2026-02-09_PEPTIDES.md`
- `FEEDBACK_2026-02-09_LISTINGS.md`
- `UPDATE_2026-02-09_PROGRESS.md`

### Message Structure:

Include these sections in messages:
1. **Date & Subject:** Clear identification
2. **Context:** Brief background or reference to previous messages
3. **Content:** Main message, research, or request
4. **Action Items:** Specific requests or next steps
5. **Status:** Current status and priorities

### Response Protocol:

When responding to messages:
1. Reference the original message filename
2. Address all questions and action items
3. Provide clear, actionable feedback
4. Suggest next steps or follow-up actions
5. Celebrate wins and acknowledge progress
